# fluttermvvmexample

## 插件描述

该插件是一个用于在VS Code中开发Flutter MVVM模式的示例插件。它提供了以下功能：

- 在指定目录下创建文件和文件夹结构
- 替换模板文件中的占位符

## 安装

1. 在VS Code中打开扩展面板（`Ctrl+Shift+X`）
2. 搜索并找到 "fluttermvvmexample" 插件
3. 点击安装按钮

## 使用

1. 打开目标项目的工作区
2. 在VS Code的命令面板中执行以下命令：`fluttermvvmexample: HelloWorld`
3. 输入类名并按回车键
4. 插件将会在指定目录下创建文件和文件夹结构，并替换模板文件中的占位符
